Что такое задача коммивояжёра?

Задача коммивояжёра — фундаментальная проблема комбинаторики, где требуется найти оптимальный маршрут через заданные точки с возвратом в исходную позицию, минимизируя общее расстояние. Применяется в логистике, ДНК-секвенировании и проектировании микросхем.

Как использовать этот инструмент:

  1. Задайте параметры графа
    Введите количество вершин (городов) в диапазоне от 3 до 10. Это определит размерность матрицы смежности.
  2. Заполните матрицу стоимостей
    Укажите расстояния между вершинами:
    • Введите целые числа в белые ячейки
    • Диагональ (расстояния до себя) автоматически заполняется "-1"
    • Используйте кнопку "Случайная матрица" для автоматического заполнения
  3. Визуализация и анализ
    После нажатия "Ввод матрицы" вы получите:
    • Графическое представление графа
    • Возможность анализа маршрутов
    • Интерактивную возможность корректировки данных

Советы для эффективной работы:

  • Для симметричных задач (A→B = B→A) заполняйте только верхнюю треугольную матрицу
  • Дважды кликните по ячейке для быстрого редактирования
  • Используйте стрелки клавиатуры для навигации между ячейками

Инструмент особенно полезен для студентов, изучающих теорию графов и специалистов по логистике.
