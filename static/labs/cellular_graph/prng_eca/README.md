Реализация веб-модуля **«Генерация псевдослучайной последовательности на элементарном клеточном автомате»** в соответствии с техническими требованиями кафедры ПИ ИИ НИЯУ МИФИ.
Модуль полностью совместим с системой **study.aia.expert**, поддерживает сохранение конфигураций, визуализацию, автоматический расчёт метрик случайности (доля 1, энтропия, серии, p-value) и адаптивный интерфейс.
